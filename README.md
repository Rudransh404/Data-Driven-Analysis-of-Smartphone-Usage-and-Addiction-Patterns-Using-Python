# Data-Driven-Analysis-of-Smartphone-Usage-and-Addiction-Patterns-Using-Python
📱 Data-Driven Analysis of Smartphone Usage and Addiction Patterns Using Python

This project performs an in-depth Exploratory Data Analysis (EDA) on a smartphone usage dataset to uncover behavioral patterns related to screen time, addiction levels, and user lifestyle factors.

Using Python and powerful data analysis libraries, the project transforms raw data into meaningful insights that highlight how smartphone usage impacts daily habits such as sleep, social media engagement, and overall digital well-being.

🔍 Key Objectives

Analyze user demographics and smartphone usage behavior
Identify patterns between screen time and addiction levels
Examine the impact of social media, gaming, and notifications
Explore relationships between screen time and sleep patterns
Detect trends and correlations using statistical analysis

🛠️ Tech Stack

Python
pandas, NumPy
matplotlib, Seaborn

📊 Key Features

Data Cleaning & Preprocessing
Distribution Analysis (Age, Screen Time)
Behavioral Analysis (Addiction Levels, Usage Patterns)
Correlation Heatmap
Insightful Visualizations (Boxplots, Histograms, Scatter Plots, Pie Charts)

💡 Key Insights

Higher screen time is strongly associated with increased addiction levels
Social media and gaming usage significantly influence addiction
Excessive smartphone usage negatively impacts sleep duration
Notifications play a role in increasing user engagement

🚀 Outcome
This project demonstrates how data science techniques can be used to analyze real-world behavioral data and generate actionable insights for improving digital well-being.
